# Parameter Sourcing

## Step-by-step checklist

1. Search the IDAES codebase.
- Look for existing parameter datasets in modular properties.
- Reuse IDs and units where possible.

2. Search grounded-docs.
- Use official examples and tables first.

3. Use external sources only if required.
- Prefer NIST, DIPPR, and standard references.

4. Use web search for placeholders.
- If no data exists, use web search for placeholder values.
- Record source and validity range.

5. Normalize units.
- Convert parameters to IDAES units.
- Record any conversions.

6. Document all sources.
- Add a short source note for each parameter group.

## Requirements

- Always capture the source and validity range.
- Track units for each parameter.
- If placeholders are used, mark them clearly.
